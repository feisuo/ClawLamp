# ClawLamp

ClawLamp 是一款内置OpenClaw的智能台灯，是您桌面的智能助理。 
  当前实现能力： 
  - LED灯控制，以来LED矩阵，实现不同的灯光效果
  - 整合了飞书，可以通过飞书和ClawLamp互动，询问工作日程，查收邮件等。
